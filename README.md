# Caddauan Financial Services Corporation

### Customer Payment Management System (CPMS)

## Overview

This repository contains the source code and related assets for the **Customer Payment Management System (CPMS)** developed for **Caddauan Financial Services Corporation (CFSC)**.

The project is intended solely for:

* Personal learning
* Portfolio demonstration
* Research and experimentation
* Salesforce development practice
* System architecture and design exercises

The CPMS platform is designed to simulate real-world financial services operations, including customer account management, payment processing, loan servicing workflows, webhook integrations, reporting, and other enterprise-level business processes.
